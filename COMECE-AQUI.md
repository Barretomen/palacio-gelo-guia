# Pacote mestre — PG Guia de Bolso

1. Abra e leia primeiro `PROMPT-MESTRE.txt`.
2. Use a pasta `REDESIGN` como referência visual obrigatória.
3. Use a pasta `PESQUISA-V2` como fonte da nova base e motor de pesquisa.
4. Aplique tudo ao projeto real já aberto.
5. Entregue um ZIP completo pronto para publicar, não apenas patches.

O ficheiro `index-profissional-demo.html` é uma referência funcional de interface. Ele não deve substituir cegamente os dados e a lógica real do projeto.
