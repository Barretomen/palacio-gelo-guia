# PG Guia de Bolso — Handoff V3

Este pacote contém a análise do projeto atual, os defeitos reproduzidos no telemóvel, requisitos de redesign responsivo, especificação do novo treino operacional e sugestões baseadas no Manual de Operações fornecido.

## Ordem obrigatória

1. Ler `PROMPT-MESTRE-ANTIGRAVITY-V3.txt` por inteiro.
2. Ler `DOCUMENTACAO/01-ANALISE-TECNICA.md`.
3. Ler `DOCUMENTACAO/02-RESPONSIVIDADE-E-DESIGN.md`.
4. Ler `DOCUMENTACAO/03-TREINO-OPERACIONAL.md`.
5. Ler `DOCUMENTACAO/04-MELHORIAS-DO-MANUAL.md`.
6. Ler `DOCUMENTACAO/05-SEGURANCA-E-PRIVACIDADE.md`.
7. Executar os testes definidos em `DOCUMENTACAO/06-TESTES-DE-ACEITACAO.md`.

## O que deve ser alterado

Trabalhar sobre a pasta `PROJETO-ATUAL`. Não criar uma demo paralela. O resultado deve ser o projeto completo, funcional, responsivo, PWA e pronto para substituir a versão atual.

## Atenção

O Manual de Operações fotografado tem data visível de **02/2013**. Nenhum procedimento interno deve ser apresentado como atual sem validação da chefia. Conteúdo operacional interno não deve ser publicado no repositório público.
